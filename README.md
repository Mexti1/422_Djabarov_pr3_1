# Приложение "Фибоначи"
## Способы отладки:
- Работа с отладчиком;
- Установка точек останова;
- Анализ значений переменных;
- Пошаговое выполнение программы.

# Приложение "Галактики"
## Способы отладки:
- Пошаговое выполнение программы;
- Точки останова;
- Проверка значений переменных и состояния объектов во время отладки;
- Работа с отладчиком.

# Приложение "Буквы"

- Пошаговое выполнение программы;
- Установка точек останова;
- Работа с отладчиком.
